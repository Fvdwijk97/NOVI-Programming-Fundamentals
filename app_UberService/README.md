# Opdracht Uber applicatie

### Doel:
Je maakt een applicatie waarin gebruikers via de commandline gebruik kunnen maken van een Uberdienst.

### Gegevens:
De gebruikers kunnen kiezen uit 3 verschillende diensten met elk een eigen prijs. Zorg dat deze diensten in een logische collectie in de applicatie staan.

De diensten zijn:
* "Uber Black" : 2.00
* "Uber Van" : 3.50
* "Uber X" : 1.50

### Opdracht:
Zorg dat de applicatie de volgende flow heeft:
* De gebruiker kan kiezen uit 1 van de drie diensten
* De gebruiker kan aangeven hoeveel km gereden moet worden
* De applicatie berekend de totale kosten
* De gebruiker krijgt te zien hoeveel de rit gaat kosten

### Bonus:
Maak een extra functionaliteit waarin de ritgeschiedenis en de dienst voorkeur van de gebruiker wordt bijgehouden.
Hiervoor kun je een woordenboek gebruiken, zoals:

USER = {"preference" : "", "history" : []}

Maak twee extra menu opties voor:
1. Waar de gebruiker diens voorkeur kan aanpassen
2. Waar de gebruiker diens geschiedenis kan inzien

Zorg ook dat je voor elke rit vraagt of de gebruiker de voorkeursdienst wilt gebruiken of een andere dienst wilt kiezen.
